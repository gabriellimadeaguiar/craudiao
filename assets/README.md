# assets

Coloque aqui a foto do Breno com o nome exato:

```
assets/breno.png
```

- Formato: PNG (ou troque a extensão no `index.html` se usar JPG).
- Recomendado: imagem mais ou menos quadrada (o rosto centralizado fica melhor,
  porque o jogo recorta num círculo).

Enquanto o arquivo `breno.png` não existir, o jogo desenha um **rosto cartoon de
fallback** automaticamente — ou seja, o jogo funciona mesmo sem a foto. Assim que
você adicionar `breno.png`, o rosto real do Breno aparece na canoa e na mesa de
cirurgia, sem precisar mexer em mais nada.
